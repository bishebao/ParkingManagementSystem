<h2>一、项目简介</h2>
基于SSM的停车管理系统

角色：管理员、用户

<h2>二、项目技术</h2>
前端 jsp、jquery、bootstrap，后端 maven、springmvc、spring、mybatis
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">

<strong>用户</strong>
登陆注册、车位查看、车主查看、个人中心、车辆记录、系统公告

<strong>管理员：</strong>
车位管理、员工管理、车主管理、车辆记录管理、系统公告管理、修改密码、个人信息、年份报表、月份报表
